CCS3 属性选择器

和jquery类似 C3中也有不少属性选择器了。

[att] 表示选择具有att属性的元素 att可以是自定义属性

  li[index]{
      color:red;
  }

[att~="val"] att属性中有用空格分开的val属性

  li[class~=fruit]{
      backgroundColor:"green";
  }

[att^="val"] att属性的值以val开头

  li[class^="icon-"]{}

[att$="val"] att属性的值以val结束

  [class$="gg]{}

[att*="val] att属性中包含字符串w

  [class*="z"]{
      fontSize:20px;
  }

可以结合以下代码进行测试

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>$别看我$</title>
    <style>
        [index]{
            color: red;
        }
        [class*="z"]{
            font-size:20px;
        }
        li[class^=icon]{
            color: blue;
        }
        li[class$=gg]{
            background-color: chocolate;
        }
    </style>
</head>
